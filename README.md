# Vårprojektet 
## Vecka 9
Vi har vallt ett UF företag som heter Kustknuten. Där dom gör anrband och så kallade apnävar som då är gjorda på gammla rep och tampar som inte används längre och på så sätt återvinner repen och tamparna.
Jag tog färgerna från deras låga och sparade dom i ett dokument sedan tog jag en primär fräg och gjorde flera olika nyanser på den och började lite på en moodpanel.
## Vecka 11
Vi gjorde klart moodpanel med gratis bilder som vi sökte up. Sen la vi in bilder på färgerna i det dokumentet som vi har gjoer alla olika nyanser av dom färgerna vi valde.
## Vecka 13
Vi började diskutera lite om hur hemsidan skulle se ut med layout och vi skisade på ett papper sedan la vi in bootstarp och färgerna i codewpen och började bolla och testa på hur det ska se ut.
